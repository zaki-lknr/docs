# html

## レスポンシブデザイン

ヘッダにこれを入れるのが基本

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```

## tag

### meta

#### 文字コード指定

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
```

サーバーの設定をいじれない場合などで。
