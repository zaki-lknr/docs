# Google Drive

## キーバインド

### 直前の操作の繰り返し

スプレッドシートとかで

```
Ctrl-y
```

### 現在日付 (スプレッドシート)

```
Ctrl + :
```

### 現在時刻 (スプレッドシート)

```
Ctrl + Shift + :
```

### コピーした行の挿入 (スプレッドシート)

機能としてないので「行の挿入」と「貼り付け」を行う必要がある。

1. まずコピーしたい行を選択してコピー
1. 選択されている状態で`Alt-i`->`r`->`r`で行の追加(選択行選択している場合、選択行数分が追加される)
1. 最後にペースト

これで「挿入」動作になる

## スライド

### 表の行・列追加

表のセル部分を右クリックするとメニューが表示されるので「行を下に挿入」など選択する。

## スプレッドシート

### 行の入れ替え

入れ替えたい対象を行選択し、左側の列番号部分を入れ替え先へD&Dする

### CSVのインポート

メニューの「ファイル」→「インポート」で「アップロード」からCSVファイルをD&Dする。  
新しいファイルにするか、現在のファイルの新規のシートか、追記するか等はアップロード後に選択できる。

### CSVテキストの貼り付け

とりあえずペーストすると、選択状態のペーストされた範囲の右下に書式貼り付けボタンが表示されるので押下、「テキストを列に分割」を選択するといい感じに表になる。

### ドロップダウンリストの作成

メニューの「挿入」→「プルダウン」から。  
画面右側にリスト定義が編集画面が表示されるので、編集画面内で定義したり、別シートの値をデータソースにするなど選択する。
