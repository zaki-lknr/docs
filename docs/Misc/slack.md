# Slack

## キーボードショートカット

Windows基準

| 項目                       | 操作                 |
| ------------------------ | ------------------ |
| ショートカット一覧の表示on/off       | `Ctrl + /`         |
| (会話ツリーなどの)右サイド画面のon/off  | `Ctrl + .`         |
| (チャンネル一覧などの)左サイドバーon/off | `Ctrl + Shift + d` |
| スレッドの表示                  | `Ctrl + Shift + t` |
| チャンネルなどの移動               | `Ctrl + k`         |
