# vi

## キーバインド

| キー                                                     | 動作             |
| ------------------------------------------------------ | -------------- |
| `:e ファイル名`                                             | ファイルオープン       |
| `:%d`                                                  | 全消し            |
| `:noh`                                                 | 検索ハイライトの解除     |
| `D`                                                    | カーソル位置から行末まで削除 |
| `dG`                                                   | カーソル行から最終行まで削除 |
| `ctrl-v`で選択開始 -> `j`/`k`で選択行指定 -> `shift-i`選択行全てで挿入モード | 矩形選択           |
| `d$`                                                   | 行末までカット        |
| `p`                                                    | ペースト           |
